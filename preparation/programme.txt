Les départements de physique et d'informatique de l'ENS Paris organisent
un stage de présentation et de formation à Python à destination des
professeurs de classes préparatoires, dans le but de les aider dans la
mise en place des nouveaux programmes qui utiliseront Python comme
langage de programmation.
 
Le but de ce stage est de présenter Python, son installation sur des
machines, de donner des éléments de programmation et calcul scientifique
avec Python, et des illustrations de son utilisation dans différents
domaines scientifiques. Le stage se déroulera sur deux jours les 15 et
16 avril 2013 à l'ENS Paris.
 
Il prévoit 4 demi journées : 1 de présentation générale, 2
d'illustrations et d'exemples du calcul scientifique, 1 d'ateliers
pratiques. Chaque session s'appuiera sur un domaine particulier
d'application, qui permettra d'introduire et d'utiliser différentes
facettes de Python et de ses modules scientifiques. Il s'agira donc d'une
présentation par l'exemple, plutôt qu'une description successive des
différents modules de Python. Les participants devront venir avec un
ordinateur portable et seront amenés à programmer des exemples courts.
 
Le programme du stage est le suivant :                                          

1ère 1/2 journée :                                                              

a) Présentation de Python. Avantages/inconvénients, contexte
d'utilisation de ce langage. Installation sur les machines des
participants. Gaël Varoquaux (INRIA Saclay) /Damien Vergnaud (ENS
informatique)

b) Quelques éléments du langage Python. Damien Vergnaud (ENS
informatique) /Gaël Varoquaux (INRIA Saclay)

                                                                                
2ème 1/2 journée :                                                              

a) Python en théorie de l'information (Shannon) Damien Vergnaud (ENS
informatique) / Gaël Varoquaux (INRIA Saclay) : mise en application du
langage Python

b) Python et phénomènes stochastiques (Mouvement Brownien) Werner
Krauth (ENS Physique) / Emmanuelle Gouillart (Saint-Gobain Recherche) :
utilisation du module NumPy (manipulation de tableaux numériques) et
matplotlib (représentation graphique)

                                                                                
3ème 1/2 journée :

a) Python pour la résolution d'équations aux dérivés partielles (loi de
Fourier) Emmanuel Dormy (ENS Physique)/ Emmanuelle Gouillart
(Saint-Gobain Recherche): utilisation des modules NumPy et matplotlib

b) Python pour l'intégration de systèmes dynamiques (Attracteur de
Lorenz)  Emmanuelle Gouillart (Saint-Gobain Recherche)/ Emmanuel Dormy
(ENS Physique) : utilisation du module SciPy

                                                                                
4ème 1/2 journée : Ateliers                                                     
                                
a) Cryptographie : Damien Vergnaud (ENS informatique)                           
b) Mécanique quantique : Werner Krauth(ENS Physique)                            
c) Hydrodynamique 2D : Emmanuel Dormy (ENS Physique)                            
d) Traitement d'images : Emmanuelle Gouillart (Saint-Gobain Recherche)     
e) Data mining: Gaël Varoquaux (INRIA Saclay)                                

  
Pour les ateliers, les participants devront indiquer lors de
l'inscription 2 ateliers souhaités parmi la liste d'ateliers proposés,
afin de pouvoir répartir les participants de manière équilibrée dans les
ateliers. 

Le nombre de participant(e)s est limité à 70.                                   

NB: Une deuxième session est envisagée durant les vacances de la
Toussaint 2013.
